# Frontend folder map

```
src/
├── app/                 # Next.js App Router (routes, layouts, metadata)
├── components/
│   ├── ui/              # shadcn/ui primitives
│   ├── layout/          # App shell providers, containers
│   ├── sections/        # Page sections (marketing blocks)
│   ├── features/        # Feature-scoped UI (booking, gallery, …)
│   ├── buttons/         # Brand button compositions
│   ├── cards/           # Card compositions
│   ├── navigation/      # Nav compositions
│   ├── footer/          # Footer compositions
│   ├── forms/           # Form compositions
│   ├── typography/      # Type scale components
│   ├── animations/      # Motion presets / wrappers
│   └── icons/           # Icon wrappers
├── lib/                 # Utils, API client, env
├── hooks/               # Shared React hooks
├── types/               # Shared TS types
├── constants/           # Non-secret site constants
├── styles/              # Extra CSS / design tokens
└── assets/              # Bundled static assets (images, fonts)
public/                  # Public static files (favicons, OG images)
```
